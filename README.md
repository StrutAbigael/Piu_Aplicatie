# Sistem de Buget Personal

## Descriere

Această aplicație reprezintă un sistem de gestionare a bugetului personal. 
Scopul aplicației este de a permite utilizatorului să își înregistreze și să își administreze veniturile și cheltuielile.

Utilizatorul poate adăuga tranzacții financiare, specificând suma, tipul tranzacției, data și o descriere. 

## Funcționalități

- Adăugare tranzacție 
- Ștergere tranzacție
- Afișare listă tranzacții
- Calcul total venituri
- Calcul total cheltuieli
- Calcul sold
- Afișare raport financiar
